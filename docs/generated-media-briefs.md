# VibeVI generated media briefs

Generated assets may be used for atmosphere, pattern, texture, illustration, and brand energy. They are not factual photos and must never represent a real business, real event, real person, real storefront, real menu item, or real partnership.

Before activating any generated asset:

1. Save the generated file under `public/media/generated/` or the specific approved media folder.
2. Record the prompt and rights notes in `data/media-manifest.json`.
3. Mark the asset as `generated_art`.
4. Keep `approved_for_business_specific_use` as `false` unless the asset is only decorative and not representing business facts.
