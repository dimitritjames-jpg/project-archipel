# Outreach tracking sheet template

Copy this table into Google Sheets, Excel, Airtable, or a CRM. Keep this as an operations tracker only; it does not change listing trust status by itself.

## Recommended columns

| Column | Purpose | Allowed examples |
|---|---|---|
| Priority | Outreach order | `1`, `2`, `3` |
| Business/place | Listing name | `Coral World Ocean Park` |
| Island | Island code/name | `STT`, `STX`, `STJ`, `WI` |
| Category | Current VibeVI category | `excursions-charters`, `indulgent-dining` |
| VibeVI route | Current public route | `/st-thomas/excursions-charters/coral-world-ocean-park` |
| Source URL | Official/public source used | URL from inventory JSON |
| Outreach channel | First channel used | `email`, `phone`, `Instagram DM`, `website form` |
| Contact name | Person contacted | blank until known |
| Contact role | Owner/manager/marketing/etc. | blank until known |
| Contact email | Email used or confirmed | blank until permission/source exists |
| Contact phone | Phone used or confirmed | blank until permission/source exists |
| First outreach date | Date sent | `2026-06-24` |
| Last touch date | Last contact attempt/reply | `2026-06-27` |
| Next follow-up date | Next scheduled action | `2026-07-02` |
| Outreach status | Current stage | see status values below |
| Corrections requested | Summary of requested changes | `Update hours`, `change category` |
| Contact permission | Display contact permission | `unknown`, `pending`, `granted`, `declined` |
| Photo rights status | Media permission | `not_requested`, `requested`, `granted`, `declined`, `needs_credit` |
| Claim interest | Claim workflow interest | `none`, `interested`, `pending_review` |
| Notes | Human notes | short, factual notes only |

## Outreach status values

- `not_started`
- `sent_initial`
- `follow_up_needed`
- `replied_needs_corrections`
- `confirmed_public_info`
- `contact_permission_granted`
- `photo_rights_requested`
- `photo_rights_granted`
- `claim_interest_recorded`
- `do_not_contact`
- `human_review_needed`

## Guardrails while using the sheet

- Do not mark a listing `verified` from this sheet alone.
- Do not change `claimed`, `premium`, `partner`, `sponsor`, or booking flags from outreach alone.
- Do not publish photos until written rights are recorded.
- Do not display new phone/email values unless permission/source status is clear.
- Keep correction notes factual and short.
