# VibeVI Agent Template — Design System

## Visual personality

Clean agent interface + real island discovery.

The interface should feel:
- calm
- immediate
- intelligent
- coastal
- approachable
- locally grounded
- easy to maintain

It should not feel:
- dark
- overly luxurious
- corporate
- like an OTA checkout
- like a tourism brochure
- like a wall of feature buttons

## Layout

- Main content max width: 1180–1240px
- Desktop header height: approximately 72–76px
- Desktop hero visual height: approximately 445–475px below header
- Desktop composer max width: approximately 760–790px
- Mobile content padding: 20–24px
- Desktop page gutter: 28–40px
- Section vertical spacing: 44–64px
- Use horizontal rails for discovery, not dense multi-row grids

## Typography

Use the project’s current clean sans-serif UI font (Geist or equivalent).

- Hero title: 54–64px desktop, 38–44px mobile
- Hero line height: 0.98–1.05
- Section title: 24–30px
- Card title: 18–22px
- Body: 15–17px
- Small label: 11–13px, uppercase only when appropriate
- Avoid a serif display system in this template

## Surfaces

- Warm off-white page canvas
- White header and cards
- Thin warm-gray borders
- Very soft shadows
- Avoid dark panels
- Avoid glassmorphism except extremely subtle composer/header effects
- Images should carry the island color; UI should stay restrained

## Agent composer

- White surface
- 22–26px radius desktop
- 18–22px radius mobile
- 1px warm-gray border
- soft large shadow
- large comfortable input zone
- teal send circle
- optional tool/voice controls only if functional
- no unnecessary inner boxes

## Chips

- white or pale neutral surface
- small shadow/border
- rounded pill
- icon + label
- 36–40px height desktop
- 34–38px mobile
- selected state: pale aqua with teal border/text
- no oversized colored category buttons

## Cards

Island cards:
- 18–20px radius
- image overlay for legible title
- three visible on desktop
- horizontal swipe on mobile

Recommendation cards:
- 16–18px radius
- image-forward
- small honest category badge
- no fake rating or availability badge
- horizontal rail

## Motion

- 150–220ms hover/focus transitions
- slight card lift only
- no large parallax
- respect reduced motion
