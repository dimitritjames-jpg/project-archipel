# Content and Trust Rules

## Agent wording

The interface may be branded `VibeVI Agent`, but v1 uses the existing search engine.

Allowed:
- What’s your island move?
- Tell VibeVI what you’re looking for.
- Get real recommendations from the USVI.
- Find beaches, boat trips, dinners, ferry routes, and rainy-day ideas.

Avoid until implemented:
- AI concierge
- AI-planned itinerary
- personalized by AI
- live availability
- instant booking
- real-time schedule
- guaranteed recommendation

## Personalization

Default recommendation rails must not imply personal behavioral personalization.

Use:
- Popular across the USVI
- Recommended places
- Local picks
- Explore next

Use `Recommended for You` only when a real query/session basis exists.

## Listings

Never invent:
- business
- owner
- rating
- review
- verification
- menu
- hours
- availability
- location
- schedule
- partnership

Public-info listings remain clearly public-info.

## Media

Use actual licensed/approved site media and honest editorial placeholders.

Do not treat generated imagery as a photograph of a named business or exact location.

## People

The Virgin Islands are a living community, not a visitor backdrop.

Use media and copy that naturally include:
- residents
- local families
- business owners
- captains/deckhands
- chefs/bartenders
- guides
- makers
- ferry workers
- visitors mixed naturally
